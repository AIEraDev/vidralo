# Changelog

All notable changes to Vidralo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-16

### Added

- Initial release of Vidralo
- Video and audio download support for 1000+ sites
- Support for YouTube, Vimeo, Twitter/X, TikTok, Instagram, and more
- Multiple format options (MP4, WebM, MP3, M4A, OPUS)
- Quality selection (4K, 1080p, 720p, 480p, etc.)
- Native desktop app for macOS (Apple Silicon & Intel), Windows, and Linux
- Auto-update functionality
- Privacy-focused: zero telemetry, no external API calls
- Homebrew Cask support for easy installation on macOS
- Cross-platform installers (DMG, MSI, EXE, DEB, RPM, AppImage)
- Bundled yt-dlp and bgutil-pot (no external dependencies required)
- Clean, intuitive user interface built with React and Tailwind CSS
- Rust-based backend for optimal performance

### Technical

- Built with Tauri 2.0 for native performance
- React 19 + TypeScript + Vite for frontend
- Zustand for state management
- GitHub Actions CI/CD for automated builds
- Multi-platform release workflow
