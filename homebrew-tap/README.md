# Vidralo Homebrew Tap

This is the official [Homebrew](https://brew.sh) tap for [Vidralo](https://github.com/AIEraDev/vidralo).

## Install

```sh
brew tap AIEraDev/vidralo
brew install --cask vidralo
```

## Upgrade

```sh
brew upgrade --cask vidralo
```

## Uninstall

```sh
brew uninstall --cask vidralo
```
